Notify customers when their Woocommerce product are updated.

== Description ==

When you update a product and you need to notify the customers about this update, with the plugin the customers will receive an email.
This is useful when you sell digital content and you get updates of your products.

Web oficial: [eggemplo](https://www.eggemplo.com/plugin/woocommerce-notify-updated-product)

== Installation ==

Add the new plugin you have received and activate it:

Go to Plugins > Add New > Upload and click the Browse... button to select the plugin's zip file, usually woo-notify-updated-product-x.y.z.zip (x, y and z are the plugin's current version numbers).

After that proceed to activate the plugin.

If you have errors, review the plugin's page located here: http://www.eggemplo.com/plugins/woocommerce-notify-updated-product for similar problems and the proposed solutions posted in the comments.
